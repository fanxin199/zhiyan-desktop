# 更新记录

## 0.2.0-alpha.1 - 2026-07-21

### 新功能

- 首次配置和基础设置中新增 DeepSeek、Kimi、通义千问与智谱 GLM 服务商选择，教师只需填写对应 API Key。
- 为四类服务预置官方兼容地址、推荐模型与模型列表，并继续支持自定义 OpenAI 兼容服务。
- 默认教学、写作和科研示例扩展到生理、病理、药理、微生物、分子生物学和免疫学等生物医学方向。

### 修复

- 修复 GLM `/v4` 接口被错误追加 `/v1` 的问题，并适配千问与 Kimi 的推理参数格式。
- 修复自由写作行内补全在切换服务商后仍访问 DeepSeek 地址的问题；非 DeepSeek 服务改用标准 Chat Completions。
- 统一课件与教材生成提示词，使其严格遵循教师选择的学科，不再默认改写为免疫学内容。
- 规范 Windows 工作区路径测试，减少不同用户目录造成的测试波动。

### 文档

- 更新 README、项目路线图和 GitHub 仓库简介，明确产品面向生物医学及相关专业大学老师。

## 0.1.0-alpha.1 - 2026-07-15

### 新功能

- 提供面向大学老师的教学、科研和能力中心三类入口，无需代码基础即可使用。
- 支持智能教案、教材编写和从 PDF/PPTX 教材生成可编辑课件及逐页讲稿。
- 支持科研文本写作、文献精读、综述撰写、自然基金撰写和科研证据追溯。
- 支持整理后的 bulk RNA-seq 与单细胞下游数据任务、分析环境预检和可复现记录。
- 提供由软件管理的私有 Python 科研分析引擎，可一键安装、修复和卸载。
- 提供自由写作台、极简 AI 对话、文件管理、任务恢复和跨模块成果衔接。

### 易用性

- 统一侧栏、工作台、搜索、最近使用和下一步任务推荐。
- 精简 AI 对话初始页面，概览和模型信息默认折叠。
- 精简自由写作台，将文件、编辑、导出和 AI 助手集中在主要工作区。
- 对教师隐藏 PATH、pip、venv、site-packages 和终端命令等技术细节。

### 安全与可靠性

- 附件内容按不可信材料隔离，不能静默改变权限或触发软件安装。
- 加固 Electron 导航、外部链接、内容安全策略、IPC 参数和工作区边界。
- 高风险文件操作和运行环境安装必须由用户明确确认。
- Windows 成品包含安装包完整性、关键运行组件和 SHA-256 校验。

### 发布说明

- 本版本为 Windows x64 内部测试版，尚未配置商业代码签名。
- 日常教学和写作功能无需 Python；科研数据分析引擎可在软件内按需安装。
