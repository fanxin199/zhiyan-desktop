# 仓库目录说明

这份说明用于判断“哪些文件属于产品，哪些文件可以忽略”。日常使用软件不需要进入或
修改任何目录。

## 产品核心

| 目录 | 用途 | 是否保留 |
| --- | --- | --- |
| `src/renderer/` | 教师看到的界面和交互 | 必须 |
| `src/main/` | 桌面应用、文件、导出、课件和 Skill 服务 | 必须 |
| `src/preload/` | 安全连接界面与桌面能力 | 必须 |
| `src/shared/` | 前后端共用的数据结构 | 必须 |
| `kun/` | Agent 运行时，负责模型、工具、Skill 和任务执行 | 必须 |
| `resources/builtin-skills/` | 随软件安装的第一方教学科研 Skills | 必须 |
| `src/asset/` | 品牌图标和界面资源 | 必须 |

## 开发与发布

| 目录 | 用途 |
| --- | --- |
| `docs/` | 产品、架构和开发说明 |
| `scripts/` | 构建、安装包和发布检查脚本 |
| `build/` | 安装包签名与平台配置 |
| `.githooks/` | 提交前、提交信息和推送前检查 |
| `.github/` | GitHub 自动测试 |
| `LICENSES/`、`THIRD_PARTY_NOTICES.md` | 第三方许可记录 |

## 自动生成，可忽略

以下目录不是源代码，删除后可由开发命令重新生成：

- `node_modules/`：安装的依赖
- `out/`：本地编译结果
- `dist/`：安装包输出
- `.worktrees/`：各功能的隔离开发工作区
- `coverage/`：测试覆盖率输出

这些目录已经写入 `.gitignore`，不会进入正式仓库。

## 已退出的方向

仓库不再包含或启动以下能力：

- DeepSeek-GUI 上游同步与品牌文件
- OpenClaw 运行时和兼容 shim
- 飞书、Lark、微信远程桥接
- 旧定时任务服务器及其 MCP 注入
- 面向程序员的代码审查、Git 分支和开发浏览器主界面

旧版本设置中的相关字段暂时只用于读取历史配置，避免升级时损坏设置；它们不会启动
服务，也不会出现在教师界面。后续完成设置迁移版本后可再删除这些兼容字段。
