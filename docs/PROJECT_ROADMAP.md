# 智研助手产品路线图

## 总目标

智研助手面向没有编码基础的高校教师。判断功能是否应该进入主界面的标准只有四项：

1. 不要求用户理解命令行、Python、Git 或模型运行时。
2. 能减少教学或科研任务中的重复劳动。
3. 关键结果可审核、可编辑、可恢复。
4. 失败时给出教师能理解和执行的处理方式。

## 已完成阶段

### 独立产品基线

- 使用全新 Git 历史和独立私有仓库，不关联 DeepSeek-GUI 上游。
- 使用独立应用标识、产品名称、图标占位稿和本地数据目录。
- 保留 MIT 来源声明，但不继承上游品牌、更新地址或发布关系。
- 恢复工作区路径边界，避免应用越权读写教师文件。

### 教学课件 Alpha

- 读取 PDF 和 PPTX 教材中的文字。
- 从 PDF 页面和 PPTX 媒体中识别候选教学图片。
- 自动去重并排除高频徽标、小图标和装饰图。
- 由教师审核图片，再自动匹配到最相关的课件页面。
- 生成可编辑 PPTX、逐页讲稿 DOCX 和可重开的项目包。
- 宽图与竖图按原比例使用不同布局，避免拉伸。
- 保留 PDF 手工裁剪，处理自动识别遗漏。

### AI 能力编排 Alpha

- 安装包内置课件、教案、题库、论文写作、文献综述、数据分析、科研绘图和文档处理 8 类第一方 Skills。
- Kun 根据对话、命令和附件类型自动激活最多 3 个相关 Skills。
- 能力中心向教师显示内置能力，项目与个人 Skills 放在高级扩展。
- Python、R、Pandoc 和 LibreOffice 等可选工具只做状态检测；安装必须经过用户确认和运行时审批。
- 内置 Skills 同步到应用私有目录，不污染 Codex、Claude 或系统全局目录。
- 已移除 OpenClaw、飞书/微信桥接和旧定时任务运行时，避免偏离教师教学科研主线。
- 教师界面只保留课程、课件、写作、文献、科研分析、文件和能力中心入口。

## 当前阶段：个人 Alpha 验证

完成标准：

- 用至少 10 份真实医学教学 PDF/PPTX 完成端到端测试。
- 首次配置、文件读取、模型失败和导出失败均有明确提示。
- Windows 安装包可在无开发环境的电脑上安装和运行。
- 教师能在不查看开发者工具的情况下完成生成、复核、导出和重开。

## 下一阶段

### P1 教学工作台

- 将教案、课程大纲、题库和课件统一为“课程项目”。
- 内置免疫学本科、研究生和国际学生三类教学模板。
- 原生导出 DOCX/PPTX，不依赖用户安装 Python。
- 增加项目自动保存、版本历史和崩溃恢复。

### P2 科研工作台

- 文献检索、证据范围和引用状态可追溯。
- 支持单细胞、空间组学、BCR/TCR 等任务的向导式入口。
- 将分析目标、QC、统计模型、图形和生物学解释保存在同一项目中。
- 高级参数默认折叠，主流程只暴露研究问题和必要选择。

### P3 可发布产品

- 完成正式品牌、图标、签名安装包和独立更新服务。
- 建立匿名崩溃报告、隐私说明、数据备份和迁移策略。
- 通过真实教师可用性测试后再扩展插件市场或开发者功能。

## 暂不进入教师主流程

- Git、worktree、hooks、MCP 配置和运行时参数只属于开发/高级设置。
- 自动把复杂整页图片重建为可编辑流程图，需单独验证准确性后再开放。
- 未验证的科研推断不得自动写成确定结论。
