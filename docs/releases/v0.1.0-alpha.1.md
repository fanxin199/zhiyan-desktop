# 智研助手 v0.1.0-alpha.1

这是智研助手的首个 Windows Alpha 预发布版本，面向希望用自然语言完成教学与科研任务、但没有代码基础的大学老师。

## 主要功能

- 智能教案、教材编写，以及从 PDF/PPTX 教材生成可编辑课件和逐页讲稿。
- 科研文本写作、文献精读、综述撰写、自然基金撰写和科研证据追溯。
- 面向整理后 bulk RNA-seq 与单细胞下游表格的科研数据分析任务。
- 自由写作台、极简 AI 对话、文件管理、任务恢复和跨模块成果衔接。
- 日常教学和写作无需 Python；科研分析环境由软件按需一键安装和管理。

## 安装与使用

1. 下载 `ZhiYan-Assistant-0.1.0-alpha.1-win-x64.exe`。
2. 按安装向导完成安装，打开智研助手。
3. 在首次配置或“设置”中填写自己的 DeepSeek API Key。
4. 从工作台选择任务，添加材料并用日常语言描述需要的结果。
