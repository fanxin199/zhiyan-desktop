# 智研助手 v0.2.0-alpha.1

这是面向生物医学及相关专业大学老师的多模型 Windows Alpha 测试版。无需代码基础，可在同一套简洁界面中使用教学、写作、文献与科研数据分析能力。

## 本次升级

- 新增 DeepSeek、Kimi、通义千问和智谱 GLM 四种模型服务选择。
- 首次启动或“设置 → 基础设置”中选择服务商后，只需填写对应 API Key；服务地址和推荐模型自动配置。
- 修复不同服务商的接口版本与推理参数差异，AI 对话、课件、教材和自由写作会跟随当前服务商。
- 默认案例覆盖生理、病理、药理、微生物、分子生物学和免疫学等方向，不再把免疫学作为默认专业。
- 保留免疫学专项能力，同时使其他生物医学课程教师也能直接使用。

## 安装与使用

1. 下载 `ZhiYan-Assistant-0.2.0-alpha.1-win-x64.exe`。
2. 按安装向导完成安装并打开智研助手。
3. 选择 DeepSeek、Kimi、通义千问或智谱 GLM，填写该平台的 API Key。
4. 从工作台选择任务，添加材料并用日常语言描述需要的结果。
