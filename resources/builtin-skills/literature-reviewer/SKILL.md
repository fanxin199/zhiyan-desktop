# 文献检索与综述

把检索和证据核实作为核心步骤。先拆分研究问题，构建主题词、同义词、时间范围、研究类型和排除条件；普通问题获得足够证据后停止，系统综述任务才扩大检索。

交付结构：

1. 检索范围与检索式。
2. 纳入证据的基本信息和证据类型。
3. 一致结论、冲突结论和研究空白。
4. 对机制、模型、样本、终点和偏倚的评价。
5. 可直接用于写作的综合段落与来源列表。

文献阅读与汇报规则：

1. 单篇 PDF 精读时，先读取题目、摘要、引言、方法、结果、讨论和图注，再生成结论；不得只基于摘要替代全文阅读。
2. 文献精读报告必须包括研究问题、实验或队列设计、关键模型、主要结果、每个主图的证据含义、局限性、替代解释和对用户课题的启发。
3. 文献汇报 PPT 任务必须先生成汇报大纲，再按背景、科学问题、研究设计、主图解读、机制整合、局限性和讨论问题组织。不要按图的数量机械生成页数。
4. 对单细胞、空间组学、BCR/TCR、肿瘤免疫和 B 细胞相关文献，主动检查亚群定义、marker 是否稳定、功能结论是否有空间、克隆性、蛋白或实验验证支持。
5. 多篇文献整理时，按主题和证据等级综合，不做逐篇摘要堆叠。明确哪些结论来自综述、原始研究、临床队列、公共数据库再分析或免疫学逻辑推断。

证据规则：

- 优先使用 PubMed、期刊官网、指南机构、临床试验注册和其他一手数据库。
- 核实标题、作者、年份、期刊、PMID、DOI 或注册号；不得生成无法验证的引用。
- 没有全文时明确写“基于摘要/元数据”，不得假装阅读了全文。
- 区分实验因果证据、临床关联、转录特征推断和免疫学逻辑推断。
- 关注发表时间与实际研究时间，不能把旧研究误写成最新发现。

若检索工具不可用，应明确说明限制并基于用户材料工作，不得用记忆补造精确引文。
