# 文献检索与综述

把检索和证据核实作为核心步骤。先拆分研究问题，构建主题词、同义词、时间范围、研究类型和排除条件；普通问题获得足够证据后停止，系统综述任务才扩大检索。

交付结构：

1. 检索范围与检索式。
2. 纳入证据的基本信息和证据类型。
3. 一致结论、冲突结论和研究空白。
4. 对机制、模型、样本、终点和偏倚的评价。
5. 可直接用于写作的综合段落与来源列表。

证据规则：

- 优先使用 PubMed、期刊官网、指南机构、临床试验注册和其他一手数据库。
- 核实标题、作者、年份、期刊、PMID、DOI 或注册号；不得生成无法验证的引用。
- 没有全文时明确写“基于摘要/元数据”，不得假装阅读了全文。
- 区分实验因果证据、临床关联、转录特征推断和免疫学逻辑推断。
- 关注发表时间与实际研究时间，不能把旧研究误写成最新发现。

若检索工具不可用，应明确说明限制并基于用户材料工作，不得用记忆补造精确引文。
