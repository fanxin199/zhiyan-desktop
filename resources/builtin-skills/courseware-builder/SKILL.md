# 教学课件设计

先识别授课对象、课时、教学目标和来源材料，再生成课件。优先使用智研助手的“课件 PPT”工作流，因为它能提取 PDF/PPTX 文字和原图、让用户复核图片，并按内容自动匹配到新页面。

输出遵循以下规则：

1. 先给出教学主线和页面蓝图，再生成逐页内容。
2. 每页只承载一个主要信息，但必须形成教师可直接投屏讲授的完整页面；正文使用完整知识陈述，禁止把教学目标、写作提示或备课建议当作课件正文。
3. 来源文件中的图必须先判断是否为有效教学图、重复徽标或装饰图；不得未经复核直接全部复用。
4. 图像按语义放到相关页面并保持宽高比，标注来源页；不得拉伸、遮挡正文或使用无法解释的图。
5. 本科教学强调概念边界和机制主线；研究生教学增加争议、实验设计和证据；国际学生提供清晰英文术语。
6. 涉及最新论文、指南或具体事实时使用可用检索工具验证，并区分事实、推断和待验证假设。
7. 最终提供每页标题、可直接展示的课堂正文、图示、讲稿、互动问题和参考答案；成品不是大纲。

若用户只在对话中提出需求，先完成可直接交给课件模块的结构化方案。不得声称已经生成或保存 PPTX，除非工具返回了真实文件路径。
