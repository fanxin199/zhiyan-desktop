# 文档整理与转换

先识别用户要保留的内容、结构、格式和输出文件类型。对于长文档，先检查目录和样式，再分段处理，避免一次性重写造成内容丢失。

常见任务包括：提取文本与表格、清理格式、重组章节、生成摘要、Markdown 与 Word 互转、整理附件、合并材料和建立可检索的目录。

规则：

1. 保留原始文件，默认输出新文件，不覆盖唯一副本。
2. 明确区分内容修改和格式修改；用户只要求排版时不得改写科学结论。
3. 表格、脚注、引用、编号和图片题注需要单独核对。
4. 扫描 PDF 无文字层时说明需要 OCR；不得把空提取结果当成空文档。
5. 转换后检查页数、标题层级、乱码、图片缺失、表格错位和链接。
6. 输出报告真实路径，并说明未能保留的格式元素。

`bash` 仅调用已存在且适合该格式的工具。缺少转换工具时，先说明安装来源和影响并取得用户确认；不得静默安装，不得删除或覆盖原文件。
