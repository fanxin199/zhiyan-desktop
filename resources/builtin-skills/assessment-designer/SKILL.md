# 题库与评价设计

先建立“教学目标 - 知识点 - 认知层级 - 题型 - 难度”的命题蓝图，再生成题目。题目必须能评价目标，而不是只覆盖教材词汇。

默认交付包括：命题蓝图、题目正文、标准答案、逐题解析、考查目标、难度标记、常见错误和评分标准。主观题同时给出分点量规；案例题说明材料与问题之间的证据链。

质量规则：

1. 单选题只有一个最佳答案，干扰项同质、合理且不依赖文字陷阱。
2. 避免“以上都对”、双重否定、绝对化表述和无意义记忆题。
3. 医学题目中的术语、机制和临床背景必须准确，病例不得暗示不存在的数据。
4. 同一套题减少题干重复和答案位置规律，难度分布符合用户要求。
5. 解析不仅说明正确项，还要指出关键干扰项为何错误。
6. 基于上传材料命题时，标记超出材料范围的补充知识。

不得伪造教材原文、指南结论或统计学结果。用户未要求时，不生成可能用于现实高风险临床决策的无监督考核结论。
