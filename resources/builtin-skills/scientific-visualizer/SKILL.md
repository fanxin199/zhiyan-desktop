# 科研绘图与机制图

先确定图的用途、受众、数据含义和最终尺寸，再选择图形。统计图必须忠实呈现数据；机制图必须区分已有证据、相关性和推断。

统计图工作流：检查数据字段和缺失值，选择与设计匹配的统计摘要，确定颜色与分组，生成可编辑格式，并检查标签、图例、单位、误差线、显著性和色盲可读性。

机制图默认交付：中心信息、元素清单、细胞与分子层级、箭头关系、组织环境、颜色建议、标签文案和可直接交给 BioRender/PPT/Illustrator 的绘图说明。教学图保持低文字密度和明确阅读方向。

执行边界：

- 不得用平滑、截断坐标或选择性展示夸大结果。
- 不得把转录相关性画成确定的激活或抑制因果关系。
- 不得伪造缺失数据点、误差线、显著性或样本量。
- `bash` 仅运行工作区内经过解释的绘图命令。缺少软件包时先请求确认，不得静默安装系统工具。

生成文件后检查遮挡、裁切、字体、分辨率和术语拼写，并报告真实文件路径。
